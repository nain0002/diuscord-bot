Problem: "mp is not defined" error
Solution: Your RAGE:MP is too old or corrupted
         Download fresh copy from rage.mp

Problem: "Cannot find module"
Solution: Run: cd C:\RAGEMP\server-files\packages\rp-server
         Then: npm install

Problem: "Database connection failed"
Solution: 1. Start MySQL service
         2. Edit .env file with correct password

Problem: Server closes instantly
Solution: Install Visual C++ Redistributables
         https://aka.ms/vs/17/release/vc_redist.x64.exe
